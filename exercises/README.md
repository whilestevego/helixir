# Helixir — Exercise Compendium 🧪

Brew Helix keybinding fluency directly in your editor. Each `.hxt` file
contains a PRACTICE section to edit and an EXPECTED section showing the goal.

## Getting Started

1. Run `helixir` from this project's root to brew the TUI
2. Open exercise files in your editor in a split pane
3. Edit the PRACTICE section to match the EXPECTED section
4. Save — the TUI detects changes and verifies automatically

## Progression

| Tier | Modules | Focus |
|------|---------|-------|
| **Apprentice** | 01-03 | Movement, selection, basic changes |
| **Adept** | 04-07 | Text objects, surround, multi-selection, search |
| **Journeyman** | 08-10 | Goto mode, space mode, unimpaired navigation |
| **Alchemist** | 11-16 | Power features, combined techniques, challenges |

## Tips

- The TUI shows instructions, commands, and hints for each exercise
- Press `Space` in the TUI to reveal hints one at a time
- Press `r` in the TUI to reset an exercise
- Use `u` to undo if you make a mistake in your editor
- Use `;` to collapse a selection back to a cursor
