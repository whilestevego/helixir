# Delimiter Objects

## PRACTICE

```js
let result = calculate(x + y * 2);
let name = "Professor Moriarty";
let config = { host: "localhost", port: 8080 };
let items = [apples, bananas, cherries];
console.log("Hello, world!");
render(createElement("div", { className: "container" }));
```

## EXPECTED

```js
let result = calculate();
let name = "";
let config = {};
let items = [];
console.log("");
render(createElement("", {}));
```
